# 💾 Agent Memory Systems (Amago Elite)

> Agentes que lembram são agentes que evoluem. A memória é o que separa um chat de um sistema de engenharia.

Esta skill gerencia como os agentes do **Amago AI Kit** armazenam e recuperam informações através de múltiplas sessões.

## 🎯 Tipos de Memória
1. **Memória de Curto Prazo**: Contexto imediato da conversa atual.
2. **Memória de Trabalho**: Arquivos temporários e estados de tarefa (task.md).
3. **Memória de Longo Prazo**: Decisões de arquitetura, preferências do usuário e aprendizados técnicos salvos.

## ✅ Checklist de Memória

- [ ] **Persistência**: O conhecimento crítico foi materializado em um arquivo `.md` ou base de dados?
- [ ] **Recuperação**: O agente conseguiu acessar informações de sessões passadas?
- [ ] **Privacidade**: Dados sensíveis estão protegidos na memória?
- [ ] **Otimização**: A memória está organizada para não estourar o limite de tokens?
