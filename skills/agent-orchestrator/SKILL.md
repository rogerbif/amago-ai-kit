# ?? Agent Orchestrator

> O cérebro descentralizado que conecta especialistas.

Esta skill gerencia a orquestração de múltiplos agentes no **Amago AI Kit**, permitindo que eles colaborem de forma autônoma.

## ?? Capacidades de Orquestração
1. **Auto-Discovery**: Varredura automática de diretórios para identificar agentes e skills disponíveis.
2. **Capability Matching**: Seleção do melhor agente baseado na tarefa e nas habilidades requeridas.
3. **Workflow Coordination**: Gerenciamento de fluxos onde um agente passa a tarefa para outro.

---

## ? Checklist de Orquestração

- [ ] **Match Perfeito**: O agente selecionado tem as skills necessárias para a tarefa?
- [ ] **Contexto Compartilhado**: A informação foi passada corretamente entre os agentes?
- [ ] **Resolução de Conflitos**: Há diretrizes claras caso dois agentes tenham opiniões divergentes?
