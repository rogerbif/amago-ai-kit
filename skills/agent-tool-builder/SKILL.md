# ?? Agent Tool Builder

> Se a ferramenta não existe, nós a construímos. A elite não se limita às capacidades padrão.

Esta skill ensina os agentes do **Amago AI Kit** a criarem suas próprias automações para resolver problemas complexos.

## ?? O Que é uma Ferramenta?
No contexto do Amago, uma ferramenta é geralmente um script (Python, JavaScript ou PowerShell) que o agente escreve e executa para realizar uma tarefa repetitiva ou complexa que o LLM não consegue fazer sozinho com precisão.

---

## ? Fluxo de Criação de Ferramenta
1. **Identificação**: Perceba que uma tarefa manual é lenta ou imprecisa.
2. **Desenvolvimento**: Escreva um script robusto com tratamento de erros.
3. **Teste**: Valide o script em um ambiente controlado.
4. **Integração**: Adicione o script à pasta `scripts/` da skill correspondente.
