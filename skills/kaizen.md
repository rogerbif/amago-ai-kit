# Amago AI Kit: Skill de Melhoria Contínua (Kaizen)

Esta habilidade foca no mindset de progresso incremental e prevenção de erros por design. No Amago AI Kit: o Kaizen é a regra de ouro: "Bom o suficiente hoje: melhor amanhã".

## Os Três Pilares Kaizen

1. **Refinamento Incremental**: Não tente a perfeição na primeira tentativa.
   - Passo 1: Faça funcionar.
   - Passo 2: Torne claro e legível.
   - Passo 3: Torne eficiente e robusto.
2. **Poka-Yoke (À Prova de Erros)**: Projete sistemas que tornem o erro impossível.
   - Utilize tipagem forte (TypeScript).
   - Valide dados na entrada do sistema.
   - Utilize Cláusulas de Guarda para evitar estados inválidos.
3. **JIT (Just-In-Time)**: Construa apenas o que é necessário agora.
   - Evite "over-engineering" para problemas imaginários do futuro.
   - Aplique o princípio YAGNI (You Aren't Gonna Need It).

## Prática Diária (Regra do Escoteiro)

- Sempre deixe o arquivo um pouco mais limpo do que você o encontrou.
- Se vir um comentário desatualizado: atualize-o ou remova-o.
- Se vir uma variável com nome confuso: renomeie-a.

## Checklist Kaizen

1. A mudança é o menor passo viável para melhorar a qualidade?
2. O design previne erros comuns através da estrutura?
3. Foi evitada a criação de funcionalidades desnecessárias para este momento?
4. O código resultante é mais simples e legível que o anterior?
