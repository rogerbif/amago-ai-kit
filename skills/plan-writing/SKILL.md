# ?? Plan Writing

> Se você falha em planejar, está planejando falhar.

Esta skill define o padrão de ouro para a escrita de planos no **Amago AI Kit**. Um plano bem escrito é o mapa que garante uma execução perfeita.

## ?? Componentes de um Plano Amago
1. **Objetivo**: O que exatamente estamos tentando alcançar?
2. **Contexto & Impacto**: Como o sistema atual será afetado? O que pode quebrar?
3. **Passos de Execução**: Lista granular de ações técnicas.
4. **Verificação**: Como provaremos que o plano funcionou?

---

## ? Checklist de Planejamento

- [ ] **Clareza**: Qualquer desenvolvedor entenderia o que será feito?
- [ ] **Granularidade**: As tarefas são pequenas o suficiente para serem validadas individualmente?
- [ ] **Impacto**: Foram identificadas possíveis mudanças que quebram o sistema (breaking changes)?
