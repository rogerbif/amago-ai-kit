# 🛡️ Amago Academy - 2026

# 👮 Auditoria de Segurança (Amago Elite)

> O Guardião nunca dorme. Segurança não é um produto, é um processo.

Esta skill define os protocolos de defesa e auditoria do **Amago AI Kit**.

## 🎯 Foco de Auditoria
1. **OWASP Top 10**: Prevenção ativa contra as ameaças mais comuns da web.
2. **Hardening**: Configurações de servidor e banco de dados para minimizar a superfície de ataque.
3. **Data Privacy**: Proteção de dados sensíveis (GDPR/LGPD).

---

## ✅ Checklist de Segurança

- [ ] **Secrets**: Chaves de API estão protegidas e fora do código?
- [ ] **Injeção**: Queries e inputs estão devidamente parametrizados/sanitizados?
- [ ] **RLS**: As regras de segurança de linha (Supabase) estão ativas?
