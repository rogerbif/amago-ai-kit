# 🧠 Skill Orchestrator (Amago Elite)

> Expandindo a consciência do Amago AI Kit.

Esta skill define como criar e orquestrar novas habilidades no padrão **Antigravity Skills**.

## 🎯 Padrão de Pasta
Toda nova Skill deve seguir esta estrutura:
```
nome-da-skill/
├── SKILL.md      # (Obrigatório) Metadados e Mapa
├── scripts/      # (Opcional) Automações
└── references/   # (Opcional) Documentação de apoio
```

## 🎯 Regras de Ouro
1. **Foco Único**: Uma Skill por domínio de conhecimento.
2. **Nomenclatura**: Use `kebab-case` para pastas.
3. **Metadados**: Sempre inclua o header YAML no `SKILL.md`.
