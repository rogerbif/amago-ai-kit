# ⚡ Supabase Automation (Amago Elite)

> O Backend as a Service definitivo para o ecossistema Amago.

Esta skill orienta a integração e automação com Supabase no **Amago AI Kit**.

## 🎯 Componentes Chave
1. **Auth**: Gerenciamento de usuários e provedores OAuth.
2. **Database (PostgreSQL)**: Queries via SDK e SQL puro.
3. **Storage**: Upload e CDN de arquivos.
4. **Edge Functions**: Lógica de servidor serverless em Deno.
5. **Realtime**: Sincronização de dados em tempo real.

## ✅ Regras de Segurança
- **RLS (Row Level Security)**: Sempre ativado. Nunca exponha tabelas sem políticas claras.
- **Service Role Key**: Use apenas no backend, nunca no cliente.
