# ?? Tailwind Patterns

> Estilização rápida: consistente e ultra-performática.

Esta skill define os padrões de uso do Tailwind CSS no **Amago AI Kit**.

## ?? Padrões de Estilo
1. **Design System First**: Use os tokens do `tailwind.config.js` (colors, spacing).
2. **Mobile First**: Aplique classes base para mobile e use prefixes (`md:`, `lg:`) para desktop.
3. **Composição sobre Duplicação**: Use componentes React para reaproveitar estilos, evite `@apply` excessivo.
4. **Variáveis Dinâmicas**: Integre com CSS Variables para temas (Dark Mode).

## ?? Mapa de Conteúdo
- `layout.md`: Flexbox, Grid e containers.
- `animation.md`: Transições e animações nativas.
- `best-practices.md`: Organização de classes e plugins úteis.
