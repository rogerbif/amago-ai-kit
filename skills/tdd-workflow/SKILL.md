# ?? TDD Workflow

> Testar não é um evento, é um estilo de desenvolvimento.

Esta skill define o fluxo de trabalho de testes no **Amago AI Kit**.

## ?? Ciclo TDD
1. **RED**: Escreva um teste que falha para a nova funcionalidade.
2. **GREEN**: Escreva o código mínimo para fazer o teste passar.
3. **REFACTOR**: Melhore o código mantendo o teste passando.
