## ?? Princípios de Design

2. **Consistência Visual**: Use os tokens do sistema rigorosamente.
3. **Feedback Imediato**: O sistema deve reagir a cada ação do usuário.
4. **Acessibilidade Nativa**: O design deve funcionar para todos.
