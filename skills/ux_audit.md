# Amago AI Kit: Skill de Auditoria de UX

Esta habilidade fornece os critérios para avaliar a usabilidade e a experiência do usuário: indo além da estética visual para garantir interfaces intuitivas e eficientes.

## Heurísticas de Usabilidade (Nielsen)

1. Visibilidade do Estado do Sistema: O usuário deve sempre saber o que está acontecendo através de feedback apropriado.
2. Correspondência com o Mundo Real: Utilize linguagem e conceitos familiares ao usuário: não jargão técnico.
3. Controle e Liberdade do Usuário: Ofereça saídas claras e funções de "desfazer" para evitar sentimentos de aprisionamento.
4. Consistência e Padrões: Siga as convenções estabelecidas para que o usuário não precise aprender novos comportamentos.
5. Prevenção de Erros: Projete a interface para evitar que erros ocorram em primeiro lugar.
6. Reconhecimento em vez de Memorização: Minimize a carga cognitiva tornando objetos e ações visíveis.

## Ergonomia Móvel

1. Zonas de Alcance: Posicione ações críticas onde o polegar do usuário alcança naturalmente.
2. Tamanho dos Alvos: Garanta que botões e links tenham tamanho suficiente para toques precisos (mínimo 44x44px).
3. Hierarquia Visual: Utilize tipografia e espaçamento para guiar o olhar do usuário para as informações mais importantes.
4. Feedback Tátil e Visual: Utilize estados de "hover": "active" e animações sutis para confirmar interações.

## Checklist de Auditoria UX

1. A interface fornece feedback imediato para as ações do usuário?
2. Os textos são claros e fáceis de entender por um leigo?
3. O usuário consegue navegar e realizar tarefas com o mínimo de esforço (cliques)?
4. A interface é acessível e funciona bem em diferentes tamanhos de tela?
