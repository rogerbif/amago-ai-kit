# 🚀 Web Performance (Amago Elite)

> Experiências instantâneas que encantam o usuário.

Esta skill define os padrões de performance Frontend para o **Amago AI Kit**, com foco em Next.js e métricas de vida real.

## 🎯 Core Web Vitals
1. **LCP (Largest Contentful Paint)**: Renderize o conteúdo principal em menos de 2.5s.
2. **FID (First Input Delay)**: Interatividade abaixo de 100ms.
3. **CLS (Cumulative Layout Shift)**: Mantenha a estabilidade visual abaixo de 0.1.

---

## ✅ Estratégias Amago
- **Image Optimization**: Use `next/image` com formatos modernos (WebP/AVIF).
- **Critical CSS**: Carregue apenas o CSS necessário para a primeira dobra.
- **Hydration Optimization**: Reduza o JavaScript no lado do cliente usando Server Components.
- **Font Loading**: Use `next/font` para evitar CLS e garantir carregamento otimizado.
